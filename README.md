# Such Skills

> A collection of Claude Code plugins for image manipulation and more.

## Installation

### Claude Code (CLI)

```bash
# Step 1: Add the marketplace
claude plugin marketplace add rajnandan1/such-skills

# Step 2: Install plugins
claude plugin install image-manipulation@such-skills
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [**image-manipulation**](image-manipulation/) | PNG-to-SVG conversion, platform-aware image resizing (40+ presets), dimension validation hook |
