# Such Skills

> A collection of Claude Code plugins for image manipulation and more.

## Installation

```bash
claude plugin marketplace add rajnandan1/such-skills
```

## Plugins

| Plugin | Description | Install |
|--------|-------------|---------|
| [**image-manipulation**](image-manipulation/) | PNG-to-SVG conversion, platform-aware image resizing (40+ presets), dimension validation hook | `claude plugin install image-manipulation@such-skills` |
