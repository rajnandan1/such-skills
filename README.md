# Such Skills

> A collection of Claude Code plugins for image manipulation, Svelte development, and more.

## Installation

### Claude Code (CLI)

```bash
# Step 1: Add the marketplace
claude plugin marketplace add rajnandan1/such-skills

# Step 2: Install plugins
claude plugin install ss-image-manipulation@such-skills
claude plugin install ss-svelte-coding@such-skills
```

## Update to latest version

```bash
claude plugin marketplace update such-skills
claude plugin update ss-image-manipulation
claude plugin update ss-svelte-coding
```

## Plugins

| Plugin                                        | Description                                                                                   |
| --------------------------------------------- | --------------------------------------------------------------------------------------------- |
| [**ss-image-manipulation**](image-manipulation/) | PNG-to-SVG conversion, platform-aware image resizing (60+ presets), lossless format conversion (PNG, JPG, WebP, TIFF, BMP, GIF, HEIC, AVIF), validation hooks |
| [**ss-svelte-coding**](svelte-coding/) | shadcn-svelte component detection and documentation for building accessible UIs in SvelteKit projects |
